# Quick Start Guide - LiveMagic Pro

## 🚀 Get Streaming in 5 Minutes

### 1. Install (2 min)
```bash
git clone https://github.com/designshop39-jpg/LiveMagic-Pro.git
cd LiveMagic-Pro
pip install -r requirements.txt
python live_streamer_enhanced.py
```

### 2. Setup Your First Stream (3 min)

#### YouTube
1. Go to https://studio.youtube.com → Create → Go Live
2. Copy your **Stream Key**
3. In LiveMagic Pro:
   - Add videos to Playlist
   - Select "YouTube" platform
   - Paste stream key
   - Click **▶ START**

#### Facebook
1. Go to facebook.com/live/producers
2. Create new live video, copy **Stream Key**
3. In LiveMagic Pro:
   - Add videos to Playlist
   - Select "Facebook" platform
   - Paste stream key
   - Click **▶ START**

#### Twitch
1. Go to twitch.tv → Settings → Channel
2. Copy **Stream Key**
3. In LiveMagic Pro:
   - Add videos to Playlist
   - Select "Twitch" platform
   - Paste stream key
   - Click **▶ START**

## 💡 Pro Tips

### Optimize Quality
- Use **1280x720 (HD)** for balanced quality/bandwidth
- Select **H.264 (CPU)** or GPU encoder for better performance
- Use **Standard** audio (192 kbps) for most streams

### Reduce CPU Usage
- Enable GPU encoding: **⚙ Settings** → Choose encoder
- Lower resolution to **854x480**
- Use **veryfast** preset

### Improve Stability
- Use **Wired Ethernet** (not WiFi)
- Close bandwidth-heavy apps
- Loop small playlist (better than large files)
- Monitor **📊 Monitor** tab

### Schedule Streams
1. Go to **🎬 Stream** tab
2. Enable "Enable Scheduler"
3. Set "Start at" time (e.g., 18:00)
4. Choose "Fixed Duration" or "30 Days"
5. Set duration (e.g., 2h 30m)
6. App will auto-start at scheduled time!

## 🎯 Common Issues

### FFmpeg Not Found
→ Settings → Download FFmpeg

### Stream Won't Connect
→ Check stream key is correct
→ Verify internet connection
→ Check firewall settings

### High CPU Usage
→ Use GPU encoder
→ Reduce resolution
→ Lower FPS to 30

### Audio Issues
→ Check system volume
→ Try different audio preset
→ Verify audio settings in Stream tab
